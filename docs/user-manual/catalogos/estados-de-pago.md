# Estados de pago

## ¿Para qué sirve?

Definir estados de los pagos (p. ej., pendiente, confirmado).

## Campos

- Código (`code`)
- Nombre (`name`)
- Activo (`is_active`)

## Filtros y búsqueda

- Código contiene
- Nombre contiene
- Fecha de creación (desde/hasta)

## Acciones

- Crear, editar, activar/desactivar, exportar

## Permisos

Prefijo: `catalogs.payment-status.*`

## Ubicación

Menú → Catálogos → Estados de pago
