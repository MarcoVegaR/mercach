# Tipos de gasto

## ¿Para qué sirve?

Clasificar gastos asociados a locales/contratos.

## Campos

- Código (`code`)
- Nombre (`name`)
- Descripción (`description`)
- Activo (`is_active`)

## Filtros y búsqueda

- Código contiene
- Nombre contiene
- Fecha de creación (desde/hasta)

## Acciones

- Crear, editar, activar/desactivar, exportar

## Permisos

Prefijo: `catalogs.expense-type.*`

## Ubicación

Menú → Catálogos → Tipos de gasto
