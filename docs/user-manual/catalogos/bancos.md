# Bancos

## ¿Para qué sirve?

Lista de bancos disponibles para asociar información financiera.

## Campos

- Código (`code`)
- Nombre (`name`)
- SWIFT/BIC (`swift_bic`)
- Activo (`is_active`)

## Filtros y búsqueda

- Código contiene
- Nombre contiene
- Fecha de creación (desde/hasta)

## Acciones

- Crear nuevo banco
- Editar banco
- Activar/Desactivar
- Exportar (CSV, XLSX, JSON)

## Permisos

Prefijo: `catalogs.bank.*` (ver Roles & Permisos)

## Ubicación

Menú → Catálogos → Bancos
