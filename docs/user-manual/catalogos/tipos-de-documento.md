# Tipos de documento

## ¿Para qué sirve?

Definir tipos de documentos y su formato.

## Campos

- Código (`code`)
- Nombre (`name`)
- Máscara (`mask`)
- Activo (`is_active`)

## Filtros y búsqueda

- Código contiene
- Nombre contiene
- Fecha de creación (desde/hasta)

## Acciones

- Crear, editar, activar/desactivar, exportar

## Permisos

Prefijo: `catalogs.document-type.*`

## Ubicación

Menú → Catálogos → Tipos de documento
