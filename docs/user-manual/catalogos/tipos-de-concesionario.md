# Tipos de concesionario

## ¿Para qué sirve?

Clasificar concesionarios por tipo.

## Campos

- Código (`code`)
- Nombre (`name`)
- Activo (`is_active`)

## Filtros y búsqueda

- Código contiene
- Nombre contiene
- Fecha de creación (desde/hasta)

## Acciones

- Crear, editar, activar/desactivar, exportar

## Permisos

Prefijo: `catalogs.concessionaire-type.*`

## Ubicación

Menú → Catálogos → Tipos de concesionario
