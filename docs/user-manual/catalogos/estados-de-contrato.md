# Estados de contrato

## ¿Para qué sirve?

Definir el estado de un contrato (p. ej., activo, finalizado).

## Campos

- Código (`code`)
- Nombre (`name`)
- Activo (`is_active`)

## Filtros y búsqueda

- Código contiene
- Nombre contiene
- Fecha de creación (desde/hasta)

## Acciones

- Crear, editar, activar/desactivar, exportar

## Permisos

Prefijo: `catalogs.contract-status.*`

## Ubicación

Menú → Catálogos → Estados de contrato
