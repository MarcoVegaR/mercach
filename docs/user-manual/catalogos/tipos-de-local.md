# Tipos de local

## ¿Para qué sirve?

Clasificar los locales según sus características.

## Campos

- Código (`code`)
- Nombre (`name`)
- Descripción (`description`)
- Activo (`is_active`)

## Filtros y búsqueda

- Código contiene
- Nombre contiene
- Fecha de creación (desde/hasta)

## Acciones

- Crear, editar, activar/desactivar, exportar

## Permisos

Prefijo: `catalogs.local-type.*`

## Ubicación

Menú → Catálogos → Tipos de local
