# Tipos de pago

## ¿Para qué sirve?

Definir los tipos de pago aceptados (p. ej., transferencia, efectivo).

## Campos

- Código (`code`)
- Nombre (`name`)
- Activo (`is_active`)

## Filtros y búsqueda

- Código contiene
- Nombre contiene
- Fecha de creación (desde/hasta)

## Acciones

- Crear, editar, activar/desactivar, exportar

## Permisos

Prefijo: `catalogs.payment-type.*`

## Ubicación

Menú → Catálogos → Tipos de pago
