# Tareas frecuentes

## Programar inspección
1. Menú → Inspecciones → Nuevo.
2. Selecciona local/espacio, fecha y objetivo.

## Registrar acta
- Abre la inspección → Registrar acta con hallazgos y evidencias.

## Cerrar hallazgos
- Marca acciones correctivas y fecha de cierre.
